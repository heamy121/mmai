# Homework 2 - Exploring Multimodal Emotion with CMU MOSEI

For homework 2, I switched to the CMU MOSEI dataset for multimodal emotion detection. The modalities present are video, audio, and text. 

In this homework, I explored different fusion points and understood the computational costs of each fusion depth. I used contrastive loss to align modalities and visualized the projections of the three modalities into the shared embedding space.
